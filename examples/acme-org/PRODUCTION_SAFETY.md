# Acme Organization Production Safety (Org Layer)

**Additional Requirements (on top of Core):**

Before any write operation in production environments:
- Type the exact phrase: **CONFIRM PRODUCTION**
- Obtain written approval from the project technical lead
- Document the change in the project TASKS.md
- Verify the change against Acme deployment checklist

All production AI-assisted changes must be logged with:
- AI model used
- Session confirmation timestamp
- Human reviewer name
