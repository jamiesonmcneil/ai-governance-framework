# Acme Organization Additional Forbidden Practices (Org Layer)

These are **additive** to the Core FORBIDDEN.md.

- Hard-coding any credentials or API keys (even temporarily)
- Using public LLMs for any work containing customer data or IP
- Committing AI-generated code without human review and tests
- Sharing internal Acme prompts or system instructions externally
- Bypassing the Session Start Protocol confirmation step
- Using localhost URLs in production configuration
